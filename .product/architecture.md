## Architecture Pattern

Next.js 16 App Router with route groups for role-based layouts. Three user roles: **owner**, **teacher**, **student**. Supabase handles auth + database (PostgreSQL with RLS policies). Stripe for payments. Mux for video hosting.

## Import/Dependency Flow

```
UI Pages → Components → lib/ helpers → Supabase Client → PostgreSQL (RLS)
                                     → Stripe SDK
                                     → Mux Player
```
